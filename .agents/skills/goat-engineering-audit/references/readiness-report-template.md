# Readiness Report Template

Use this shape when the user asks for a formal audit or readiness call:

## Conclusion

- score
- whether to start formal review
- whether feature expansion should wait

## Findings

For each finding include:

- Finding
- Severity
- Why it matters
- Evidence
- Minimal fix
- Files / workflows / tests affected
- What blocks calling this ready

## Closeout

- Evidence already supporting readiness
- Remaining gaps
- External blockers
- Smallest next closure slice
